# Software Requirements Specification (SRS)

## Functional Requirements
- **Game Loop**: Identify 60 FPS loop with delta time.
- **Game Mechanics**: Tetromino rotation (SRS), line clearing, gravity, ghost piece.
- **Input**: Keyboard support (Arrow keys, Space, Shift, Z/X).
- **Audio**: Sound effects for rotation, lock, clear.

## Non-Functional Requirements
- **Performance**: Maintain 60 FPS on average hardware.
- **Compatibility**: Chrome, Firefox, Safari (latest versions).
- **Code Quality**: Strict Type checking, 90%+ Test coverage.

## User Stories
- As a player, I want to play Tetris in the browser without downloading plugins.
- As a developer, I want to build the project with `npm install && npm start`.
